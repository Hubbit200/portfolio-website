Projects

Spotify History Analyzer

I decided to make this after noticing the lack of any visible statistics about my Spotify listen history. Initially a Java-based downloadable, I then ported it to Javascript and made a web version to make it easier to use. All the data processing is client-side to maintain privacy.

Github Try it out!

Chat Analyzer

Wanting a way to see statistics about messages sent in various apps, I made this program which shows information about chats imported from WhatsApp, Telegram, or Instagram - such as the number of messages per user, over time, and the most common words used.

Github

This Portfolio Website
